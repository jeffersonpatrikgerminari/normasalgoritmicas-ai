# normasalgoritmicas.ai

Site institucional (Next.js + Tailwind) com versão PT/EN.

## Como rodar
1. Instale dependências:
   ```bash
   npm install
   ```
2. Rode:
   ```bash
   npm run dev
   ```
3. Acesse:
   http://localhost:3000

## Publicar na Vercel
- Suba este projeto no GitHub
- Import na Vercel e Deploy
- Conecte o domínio normasalgoritmicas.ai
